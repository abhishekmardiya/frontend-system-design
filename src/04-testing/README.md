# Testing

From this folder:

```bash
npm install
npm test
```

Runs Jest against `app.test.js` (sorting helper in `app.js`).

## Files

| File           | Role                             |
| -------------- | -------------------------------- |
| `app.js`       | Sample function under test       |
| `app.test.js`  | Jest specs                       |
| `package.json` | Local deps and `npm test` script |
