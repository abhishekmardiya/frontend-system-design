# Short polling

From the repository root: `npm run start:short-polling` — **http://localhost:3000** (port set in `index.js` via `app.listen`).

## Files

| File         | Role                          |
| ------------ | ----------------------------- |
| `index.js`   | Express server and `/getData` |
| `index.html` | Demo page                     |
