# Long polling

From this folder: `npm install` then `npm start` — **http://localhost:3000** (port set in `index.js` via `app.listen`).

## Files

| File         | Role                                      |
| ------------ | ----------------------------------------- |
| `index.js`   | Express server, `/getData`, `/updateData` |
| `index.html` | Demo page                                 |
